# Ecommerce Scraper MXBR

## Project Documentation

This project is designed to scrape ecommerce data from various online platforms to facilitate data analysis and product monitoring.

### Features
- Scrapes product information such as names, prices, and availability.
- Supports multiple ecommerce websites.
- Outputs data in CSV format.

### Requirements
- Python 3.x
- BeautifulSoup4
- Requests
- Pandas

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/JacAkon/ecommerce-scraper-mxbr.git
   ```
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

### Usage
To run the scraper, execute the following command:
```bash
python scraper.py
```
